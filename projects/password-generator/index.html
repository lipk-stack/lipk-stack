Is this safe?

Yes. Passwords are generated with crypto.getRandomValues — the same cryptographically secure randomness browsers use for encryption — and never leave your device or get logged. For maximum safety, store the result in a password manager.

Frequently asked questions

Are these passwords really random and secure?

Yes. Passwords are generated with crypto.getRandomValues, the same cryptographically secure randomness browsers use for encryption. They are created on your device, never sent anywhere and never logged, so each one is unique to you.

How long should my password be?

Aim for at least 16 random characters, or a passphrase of 4 or more random words, for strong everyday security. Longer is stronger — the strength meter shows the entropy so you can see the effect of adding length or character types.

What is a passphrase and is it safer?

A passphrase strings together several random words, like “river-camera-7-mellow”. It can be just as strong as a random string but far easier to type and remember. For accounts you log into by hand, a passphrase is often the better choice.
