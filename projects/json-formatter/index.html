JSON formatter & validator

Paste JSON to beautify, minify or validate it. Invalid JSON shows the exact error with line & column. Nothing is uploaded — it all runs in your browser and autosaves locally.

Format, validate & debug JSON safely

Great for cleaning up an API response, validating a config file, spotting a missing comma or bracket, or shrinking JSON before pasting it somewhere with a size limit. Validation uses your browser's own JSON engine, so it matches the JSON spec exactly. Because it runs 100% on your device, you can safely format sensitive payloads (tokens, customer data) — nothing is ever uploaded to a server, unlike many online JSON tools.
