# Image Compressor & Resizer

A 100% client-side image compressor and resizer. Drop in JPG/PNG/WebP files,
choose an output format, quality and max width, and download smaller images.
Everything runs in the browser via the Canvas API — **no upload, no server, no
dependencies**.

- Live URL (once Pages is enabled): https://lipk-stack.github.io/lipk-stack/image-compressor/
- Files: `index.html`, `app.js`, `config.js` (+ shared `../assets/site.css`, `../assets/support.js`)

## Monetization
See `config.js`. Optional, zero-cost to set up, hidden until you fill it in:
- **Tip jar** (Ko-fi / Buy Me a Coffee)
- **Pro upsell** (e.g. batch compression + ZIP export, sold via Gumroad)
- **Affiliate links** (cloud storage, stock photos)

These require the owner's identity + payout details and cannot be automated.

## Privacy
Images never leave the device. The "nothing is uploaded" angle is the main
differentiator versus typical online compressors.
